Detailed Description

PktStatsCollector collects the statistics for the number of times when there was need to flush an outdated packet in the hardware queues in NIC.

The Ath9k chip doesn't support on demand flushing of hardware queues. But there might be scenarios (e.g. for CAMs) where outdated packets, which could not make their way due to heavy channel load need to be flushed. This implementation is specific to Ath9k chipset, with modified Linux Kernel 3.18.
