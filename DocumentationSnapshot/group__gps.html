Detailed Description

The GPS module can connect to GPSd to measure real GPS data.

If such data is available, it publishes it via zmq.

Feature Highlight:
Alternatively, the GPS module can simulate GPS data using one of two modes: It either simulates a vehicle driving towards north on a straight line or uses previously recorded GPS traces. The GpsConfig allows to specify whether or not to simulate and which simulation mode to use.
